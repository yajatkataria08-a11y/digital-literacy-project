## Project Overview
This repository contains my work as a Student Digital Ambassador. It documents my journey through five modules of Digital Literacy, from professional branding to cybercrime awareness.

## Module Summaries
* **Task 1: Infographic** - A visual guide on digital tools, safety, and etiquette.
* **Task 2: Digital Portfolio** - Professional profiles set up on GitHub, LinkedIn, and Kaggle.
* **Task 3: Platforms** - Coding practice on HackerRank and a Digital Literacy Quiz.
* **Task 4: Email Etiquette** - Professional communication drafts and social media guidelines.
* **Task 5: Cybercrime** - A case study on online fraud and a prevention checklist.
